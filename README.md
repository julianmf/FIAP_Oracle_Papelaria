Sistema de Gestão para Papelaria

Este projeto representa o modelo de dados completo de uma papelaria, contemplando os principais processos operacionais do negócio, desde o cadastro de produtos até vendas, controle de estoque e gestão fiscal. O objetivo é fornecer uma base sólida e normalizada para suportar sistemas de gestão comercial e administrativa.

O sistema permite o cadastro estruturado de produtos, organizados por espécie, grupo, marca e cor, garantindo padronização e rastreabilidade. O controle de estoque é feito por loja, setor e produto, com informações de quantidade, custo, lote, data de fabricação e localização física.

A solução cobre todo o fluxo de compras, incluindo ordens de compra, fornecedores, notas fiscais de entrada e itens recebidos, bem como o fluxo de vendas, com pedidos, itens de pedido, emissão de notas fiscais de venda e cálculo de impostos. Também há suporte à gestão de clientes, funcionários e fornecedores, utilizando uma modelagem unificada de pessoas físicas e jurídicas, com controle de contatos e endereços.

O modelo foi desenvolvido em Oracle Database, com uso de chaves primárias e estrangeiras, constraints, triggers, sequences e regras de integridade, garantindo consistência, integridade referencial e aderência às regras de negócio. É um projeto indicado para fins acadêmicos, didáticos ou como base para sistemas ERP de pequeno e médio porte no segmento de papelaria.
